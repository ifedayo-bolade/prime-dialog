(v1.0.0)
- Initial release.

(v1.0.1)
- Removed unused resources
- Removed 'setWindowAnimationEnabled(boolean)'
- Added 'setDontShowAgainColor()' and 'setDontShowAgainColorRes()'.
- Added built-in window enter and exit animation style resources:
    'PrimeDialog.WINDOW_ANIMATION_FADE'
    'PrimeDialog.WINDOW_ANIMATION_FLASH' -> (Used as default)
    'PrimeDialog.WINDOW_ANIMATION_SLIDE'
to be used with 'setWindowAnimation(styleRes)'.
- Fixed title typeface not getting applied with 'setTitleTypeface()' and 'setTitleTypefaceRes()'.

(v1.0.2)
- Renamed 'setMessageMargin()' to 'setMargin()'
- Ability to set custom layout/view margin by calling 'setMargin()'
